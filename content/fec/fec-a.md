## 符号表

| 符号          | 名称              | 说明                                                                         |
| ------------- | ----------------- | ---------------------------------------------------------------------------- |
| $d_i$         | 消息符号          | 待编码的消息                                                                 |
| $\bm{d}$      | 消息符号          | 待编码的消息组成的向量                                                       |
| $p_i$         | 校验符号          | 编码添加的校验符号                                                           |
| $c_i$         | 码字符号          | 编码之后的消息                                                               |
| $\bm{c}$      | 码字向量          | 编码之后的消息组成的向量                                                     |
| $\bm{e}$      | 错误向量          | 在消息传输过程中出现的错误                                                   |
| $\Sigma$      | 符号表            | 消息和码字的符号表                                                           |
| $n$, $k$      | 码长              | $k$ 个符号长的消息序列编码成 $n$ 个符号长的码字                              |
| $r$           | 校验码长          | $r = n - k$                                                                  |
| $s$           | 缩短码长度        | 缩短码缩短的长度                                                             |
| $R$           | 码率              | $R = k / n$                                                                  |
| $\bm{G}$      | 生成矩阵          | 线性分组码的生成矩阵，或称编码矩阵                                           |
| $\bm{H}$      | 校验矩阵          | 线性分组码的校验矩阵，或称解码矩阵                                           |
| $\bm{S}$      | 特征值            | 或称校验值                                                                   |
| $GF(q)$       | 伽罗华域          | 或称为有限域                                                                 |
| $p$           | 特征值            | $GF(p)$ 的特征值（Characteristic），质数                                     |
| $m$           | 扩域次数          | $GF(p^m)$ 中扩域次数，正整数                                                 |
| $q$           | 域的阶            | $q = p^m$，域 $GF(q)$ 的阶数                                                 |
| $f(x)$        | 多项式            | $f(x)$ over $GF(q)$，$GF(q)$ 上的多项式 $f(x)$                               |
| $m(x)$        | 最小多项式        |                                                                              |
| $p(x)$        | 本原多项式        | 通常用于构造扩展域 $GF(p^m)$                                                 |
| $g(x)$        | 生成多项式        | 用于构造多项式码                                                             |
| $d(x)$        | 信息多项式        | 待编码的消息作为系数构成的多项式                                             |
| $c(x)$        | 码字多项式        | 编码之后的消息为系数构成的多项式                                             |
| $e(x)$        | 错误多项式        | 在消息传输过程中出现的错误为系数构成的多项                                   |
| $z(x)$        | 接收多项式        | 译码器接收到的消息为系数构成的多项                                           |
| $\alpha$      | 本原元素          | 伽罗华域 $GF(q)$ 的本原元素                                                  |
| $a$, $b$, $c$ | 域元素            | 伽罗华域 $GF(q)$ 的元素                                                      |
| $d$           | 汉明距离          |                                                                              |
| $d_{min}$     | 最小汉明距离      |                                                                              |
